# NuQu
Physics 765, S26: Quantum Algorithms and Error Correction. Final Project: a tool bridging the gap between quantum chemistry toolkits and nuclear physics
I propose to develop a software toolkit that bridges quantum chemistry quantum-compilation tools with the unique requirements of nuclear many-body physics. The tool will take nuclear Hamiltonians—featuring multiple fermion species and higher-body interactions—and translate them into qubit representations compatible with modern quantum-compilation frameworks such as Qualtran or PyLIQTR. This will create a reproducible, full-stack pipeline for exploring how fault-tolerant quantum computers could be used for realistic nuclear simulations.
